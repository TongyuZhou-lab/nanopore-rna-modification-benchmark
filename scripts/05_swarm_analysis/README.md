# SWARM deep analysis and threshold-scan plotting

This folder contains two scripts for SWARM-related analysis in the m6A benchmark.

## Scripts

### 1. `swarm_deep_analysis_cli.py`

This script performs the main SWARM/m6Anet deep analysis from site-level input files.

It can:

* build a labeled common-site evaluation table
* scan SWARM probability thresholds
* scan SWARM stoichiometry thresholds
* scan joint probability + stoichiometry thresholds
* compare SWARM, m6Anet, their intersection, and their union
* optionally compare f5c-SWARM and nanopolish-SWARM results

Main outputs include:

```text
base_eval_table.with_SWARM_m6Anet.tsv
SWARM_probability_threshold_scan.tsv
SWARM_stoichiometry_threshold_scan.tsv
SWARM_joint_probability_stoichiometry_threshold_scan.tsv
SWARM_m6Anet_strategy_metrics.tsv
SWARM_m6Anet_labeled_site_categories.tsv
README.txt
```

Example:

```bash
python swarm_deep_analysis_cli.py \
  --common-bed path/to/common_evaluation_universe.bed \
  --positive-bed path/to/positive_reference_sites.bed \
  --exclude-bed path/to/exclude_reference_sites.bed \
  --swarm-scorebed path/to/SWARM.score.bed \
  --swarm-stoich-table path/to/SWARM_stoichiometry_table.tsv \
  --m6anet-scorebed path/to/m6Anet.score.bed \
  --outdir path/to/swarm_deep_analysis_output
```

Optional nanopolish-SWARM comparison:

```bash
python swarm_deep_analysis_cli.py \
  --common-bed path/to/common_evaluation_universe.bed \
  --positive-bed path/to/positive_reference_sites.bed \
  --exclude-bed path/to/exclude_reference_sites.bed \
  --swarm-scorebed path/to/f5c_SWARM.score.bed \
  --swarm-stoich-table path/to/f5c_SWARM_stoichiometry_table.tsv \
  --m6anet-scorebed path/to/m6Anet.score.bed \
  --swarm-nanopolish-sitelevel path/to/nanopolish_SWARM_sitelevel.tsv \
  --outdir path/to/swarm_deep_analysis_output
```

Important default cutoffs:

```text
SWARM probability cutoff: 0.9972
SWARM stoichiometry cutoff: 0.1
m6Anet cutoff: 0.9
exclude buffer: 2 bp
```

These can be changed using:

```bash
--swarm-prob-cutoff 0.9972
--swarm-stoich-cutoff 0.1
--m6anet-cutoff 0.9
--exclude-buffer 2
```

### 2. `plot_swarm_threshold_scan.py`

This script plots the final unified-style figures from the output tables generated by `swarm_deep_analysis_cli.py`.

It does not rebuild the evaluation table. It only reads the threshold-scan and strategy-metric tables, then produces full and clean figure versions.

Input files expected in the analysis output directory:

```text
SWARM_probability_threshold_scan.tsv
SWARM_stoichiometry_threshold_scan.tsv
SWARM_joint_probability_stoichiometry_threshold_scan.tsv
SWARM_m6Anet_strategy_metrics.tsv
```

Example using an input directory:

```bash
python plot_swarm_threshold_scan.py \
  --indir path/to/swarm_deep_analysis_output \
  --outdir path/to/swarm_threshold_scan_figures
```

Example using explicit input files:

```bash
python plot_swarm_threshold_scan.py \
  --prob-tsv path/to/SWARM_probability_threshold_scan.tsv \
  --stoich-tsv path/to/SWARM_stoichiometry_threshold_scan.tsv \
  --joint-tsv path/to/SWARM_joint_probability_stoichiometry_threshold_scan.tsv \
  --strategy-tsv path/to/SWARM_m6Anet_strategy_metrics.tsv \
  --outdir path/to/swarm_threshold_scan_figures
```

Main outputs:

```text
full/
clean/
tables/
```

The `full/` figures contain titles, legends, and axis labels.
The `clean/` figures remove titles and legends, while retaining axes and tick labels for manuscript assembly.

Common style options:

```bash
--fig-width 3.0
--fig-height 3.0
--tick-labelsize 12
--axis-labelsize 9
--legend-size 7
--dpi 300
```

## Recommended workflow

Run the analysis script first:

```bash
python swarm_deep_analysis_cli.py \
  --common-bed path/to/common_evaluation_universe.bed \
  --positive-bed path/to/positive_reference_sites.bed \
  --exclude-bed path/to/exclude_reference_sites.bed \
  --swarm-scorebed path/to/SWARM.score.bed \
  --swarm-stoich-table path/to/SWARM_stoichiometry_table.tsv \
  --m6anet-scorebed path/to/m6Anet.score.bed \
  --outdir path/to/swarm_deep_analysis_output
```

Then generate the final unified figures:

```bash
python plot_swarm_threshold_scan.py \
  --indir path/to/swarm_deep_analysis_output \
  --outdir path/to/swarm_threshold_scan_figures
```

## Notes

`swarm_deep_analysis_cli.py` is used for analysis and table generation.

`plot_swarm_threshold_scan.py` is used for final figure generation.

For manuscript figures, the outputs from `plot_swarm_threshold_scan.py` are recommended because they follow a unified plotting style.
